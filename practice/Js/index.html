Let vs Var vs Const

VAR is used for variable declaration in javascript. Variables created by var are function scoped.Var support hoisting

LET allows you to declare variables that are limited to the scope of a block statement. Let does not support hoisting

Variable declared by let cannot be redeclared and must be declared before use.

CONST allows you to declare variables that are limited to the scope of a block statement. Const Variables cannot be redeclared or reassigned.

Hoisting

JavaScript Hoisting refers to the process whereby the interpreter appears to move the declaration of functions, variables or classes to the top of their scope, prior to execution of the code.

Hoisting allows functions to be safely used in code before they are declared.
